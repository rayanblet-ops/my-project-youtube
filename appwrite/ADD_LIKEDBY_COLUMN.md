# Добавление колонки likedBy для системы лайков

## Что нужно сделать:

В Appwrite Dashboard нужно добавить новую колонку `likedBy` в коллекцию `videos`.

### Шаги:

1. Откройте Appwrite Dashboard → Databases → `main` → `videos`
2. Перейдите в раздел **Columns** (или **Attributes**)
3. Нажмите **"Create column"**
4. Заполните форму:

#### Вариант 1: Использовать тип String (JSON) - Рекомендуется

   - **Key**: `likedBy`
   - **Type**: `String`
   - **Size**: `5000` ⚠️ **ВАЖНО: Используйте размер 5000, а не 500!** (достаточно для хранения массива userId в формате JSON)
   - **Required**: ❌ (выключите)
   - **Array**: ❌ (выключите)
   - Нажмите **Create**

⚠️ **Если у вас уже создана колонка с размером 500:**
1. Удалите старую колонку `likedBy` (если возможно)
2. Создайте новую с размером **5000**
3. Или обновите существующую колонку, увеличив размер до 5000

#### Вариант 2: Использовать тип Array (если доступен)

Если в вашей версии Appwrite есть тип **Array**:
   - **Key**: `likedBy`
   - **Type**: `Array` (если доступен в выпадающем списке)
   - **Required**: ❌
   - **Array**: ✅ (включите, если Type = Array)
   - Нажмите **Create**

**Рекомендую использовать Вариант 1 (String)** - он более универсальный и точно будет работать.

### Важно:

После добавления колонки:
- Все новые видео будут автоматически создаваться с пустым массивом `likedBy: []`
- Для существующих видео можно вручную установить `likedBy: []` или оставить пустым
- Система лайков будет работать автоматически

## Проверка:

После добавления колонки:
1. Обновите страницу в браузере
2. Попробуйте поставить лайк на видео
3. Обновите страницу - лайк должен сохраниться
4. Попробуйте убрать лайк - должно работать
