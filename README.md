# YuTube

Видео-платформа на React с интеграцией Appwrite.

## Технологии

- React 18
- TypeScript
- Tailwind CSS
- React Router v6
- Appwrite (Database, Storage, Auth)

## Установка

```bash
npm install
npm run dev
```

## Функции

- Авторизация (email/password, вход по имени пользователя)
- Загрузка видео и изображений
- Просмотр видео
- Комментарии
- Управление каналом
- История просмотров
- Система лайков
